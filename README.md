\# NASA-GeneLab-Training



Practice scripts and Jupyter Notebooks from NASA GeneLab GL4U training module.



This repository contains my practice scripts and notebooks based on the open-source training provided by NASA GeneLab (GL4U Module).



\## Skills \& Modules Covered



During this training pipeline, I have successfully implemented and practiced the following bioinformatics workflows:



\* \*\*01. Introduction to Jupyter \& Linux:\*\* Setting up environment, understanding command-line interface basics for high-throughput sequencing data.

\* \*\*02. RNA-Seq Processing Pipeline:\*\* Handled end-to-end processing pipelines for transcriptomics data using cloud infrastructure.

\* \*\*03. Statistical Analysis with R:\*\* Data visualization, differential gene expression (DGE) analysis concepts, and plotting genomic data using R.

\* \*\*04. Isoform-Level Analysis:\*\* Exploring advanced RNA-Seq processing pipelines, identifying gene isoforms, and quantification.



\## Tech Stack Used



\* \*\*Languages:\*\* Python, R, Bash (Linux)

\* \*\*Platforms:\*\* Google Colab, Jupyter Notebooks, GitHub Desktop

\* \*\*Domain:\*\* Transcriptomics / Functional Genomics
